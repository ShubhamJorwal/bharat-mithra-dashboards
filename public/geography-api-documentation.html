üìñ Overview

The GENOME Geography API provides comprehensive access to India's administrative divisions data. This API is designed for government applications and includes accurate Census 2011 data.

Base URL

https://your-domain.com/api/v1/geography

Authentication

Most endpoints are public and don't require authentication. Write operations (POST, PUT, DELETE) may require admin authentication.

üèõÔ∏è States / Union Territories API

Method Endpoint Description
GET /geography/states List all states with pagination and filters
POST /geography/states Create a new state
GET /geography/states/:id Get a single state by UUID
GET /geography/states/code/:code Get state by 2-letter code (e.g., KA, MH)
PUT /geography/states/:id Update a state
DELETE /geography/states/:id Soft delete a state
GET /geography/states/:id/districts Get all districts for a state
GET /geography/states/:id/summary Get summary statistics for a state
