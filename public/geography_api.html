Universal Search

GET /geography/search

Search across all location types - states, districts, taluks, gram panchayats, and villages.

Query Parameters

ParameterTypeRequiredDescription
qstringYesSearch query (minimum 2 characters)
typestringNoFilter by type: state, district, taluk, gram_panchayat, village
state_iduuidNoLimit search to a specific state
limitintNoMaximum results (default: 20, max: 100)

Common Filter Parameters

All list endpoints support the following common filter parameters:

Pagination

ParameterTypeDefaultDescription
pageint1Page number (starts from 1)
per_pageint20Items per page (max: 100)

Sorting

ParameterTypeDescription
sort_bystringField to sort by (name, code, created_at, sort_order, population)
sort_orderint0 = DESC (descending), 1 = ASC (ascending)

Date Range

ParameterTypeFormatDescription
date_fromstringYYYY-MM-DDStart date for filtering
date_tostringYYYY-MM-DDEnd date for filtering
date_fieldstring-Which date field to filter (created_at, updated_at)

Status

ParameterTypeDescription
is_activeboolFilter by active status (true/false or 1/0)
include_inactiveboolInclude inactive records

Example Filter Usage

# Get all districts in Maharashtra, sorted by name ascending
GET /geography/districts?state_id=uuid&sort_by=name&sort_order=1

# Get villages created in last month, paginated
GET /geography/villages?date_from=2024-12-01&date_to=2024-12-31&page=1&per_page=50

# Search for locations containing "mumbai"
GET /geography/search?q=mumbai&type=district

# Get all active taluks in a district
GET /geography/taluks?district_id=uuid&is_active=true

Response Status Codes

CodeDescription
200Success - Request completed successfully
201Created - Resource created successfully
400Bad Request - Invalid parameters or request body
404Not Found - Resource not found
500Internal Server Error - Something went wrong

Error Response Format

{
  "success": false,
  "message": "Resource not found",
  "error": "state with id 'xyz' not found"
}
