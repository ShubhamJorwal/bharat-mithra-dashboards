Overview

The Services API provides comprehensive endpoints to manage government services, categories, dynamic forms, and FAQs. Services are organized into categories (with support for subcategories) and can have associated forms for application submission.

Base URL

https://api.genome.gov.in/api/v1

Key Features

  • Hierarchical category structure with parent-child relationships
  • Service filtering by category, popularity, and featured status
  • Full-text search across services
  • Dynamic form schemas with versioning
  • Bilingual support (English and Hindi)
  • Available locations for each service

Restrictions

Cannot delete categories that have subcategories or services. Move or delete child items first.

Form Endpoints

Form Versioning

Forms are versioned automatically. Creating a new form for a service increments the version and deactivates previous versions. Only one form can be active per service at a time.
