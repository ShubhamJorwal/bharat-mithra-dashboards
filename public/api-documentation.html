Get all user documents - Coming soon

Database Schema

states_master
Master table for all Indian states and union territories
  • codeVARCHAR(5)PRIMARY KEY
  • nameVARCHAR(100)NOT NULL
  • name_hindiVARCHAR(100)NOT NULL
  • typeVARCHAR(20)state | union_territory
  • capitalVARCHAR(100)
  • regionVARCHAR(50)north|south|east|west|central|northeast
  • is_activeBOOLEANDEFAULT true
  • sort_orderINT
Records: 28 States + 8 Union Territories = 36 total
users
Main users table - PARTITIONED BY state_code for 900M+ users
  • idUUIDPRIMARY KEY
  • mobileVARCHAR(15)UNIQUE per state
  • mobile_verifiedBOOLEAN
  • emailVARCHAR(255)INDEXED
  • email_verifiedBOOLEAN
  • password_hashVARCHAR(255)
  • full_nameVARCHAR(255)NOT NULL
  • full_name_hindiVARCHAR(255)
  • date_of_birthDATE
  • genderVARCHAR(10)male|female|other
  • profile_photo_urlVARCHAR(500)
  • aadhaar_number_hashVARCHAR(64)INDEXEDHashed for security
  • aadhaar_verifiedBOOLEAN
  • pan_numberVARCHAR(10)
  • pan_verifiedBOOLEAN
  • address_line1VARCHAR(255)
  • address_line2VARCHAR(255)
  • cityVARCHAR(100)
  • district_idUUIDFK districts
  • state_codeVARCHAR(5)PARTITION KEY
  • pincodeVARCHAR(6)
  • statusVARCHAR(20)active|inactive|suspended|deleted
  • kyc_statusVARCHAR(20)pending|partial|verified|rejected
  • preferred_languageVARCHAR(10)en|hi
  • last_login_atTIMESTAMP
  • login_countINT
  • created_atTIMESTAMP
  • updated_atTIMESTAMP
  • deleted_atTIMESTAMPSoft delete
Partitions: 37 partitions (users_mh, users_up, users_tn, etc.)
Capacity: ~25 million users per partition = 900M+ total
user_documents
User uploaded documents (Aadhaar, PAN, etc.)
  • idUUIDPRIMARY KEY
  • user_idUUIDFK users
  • user_state_codeVARCHAR(5)
  • document_typeVARCHAR(50)aadhaar_front|pan_card|voter_id|...
  • document_numberVARCHAR(100)
  • document_nameVARCHAR(255)
  • issue_dateDATE
  • expiry_dateDATE
  • file_urlVARCHAR(500)NOT NULL
  • file_typeVARCHAR(20)pdf|jpg|png
  • file_size_bytesBIGINT
  • verification_statusVARCHAR(20)pending|verified|rejected|expired
  • verified_byUUIDAdmin who verified
  • uploaded_byVARCHAR(20)user|agent|admin
  • ocr_dataJSONBExtracted OCR data
otps
OTP storage for authentication
  • idUUIDPRIMARY KEY
  • mobileVARCHAR(15)INDEXED

  • purposeVARCHAR(20)login|register|reset_password|...

  • verified_atTIMESTAMP
